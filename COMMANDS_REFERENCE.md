## 🚀 Quick Start

1. **Start RAGS:**
   ```bash
   cd backend && npm start
   cd desktop && npm run dev
   ```

2. **Open Browser:**
   ```
   http://localhost:1420
   ```

3. **Click Microphone**

4. **Say:**
   ```
   "Hey RAGS, search Python"
   ```

5. **Listen to Response!** 🔊
