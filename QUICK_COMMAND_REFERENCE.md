## 🎯 TIPS

### Best Practices
```
✅ Speak clearly
✅ Use natural language
✅ Hindi/English/Hinglish - all work
✅ Wait for response before next command
✅ Listen to TTS - RAGS will speak back!
```

### What Works Best
```
✅ Short, clear commands
✅ Natural phrases
✅ Hinglish is perfectly fine
✅ One command at a time
```

### What to Avoid
```
❌ Very long sentences
❌ Multiple commands at once
❌ Background noise while speaking
❌ Speaking during TTS playback
```
