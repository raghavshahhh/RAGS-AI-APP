## 🎯 How to Use

### Desktop App

1. **Start RAGS**
   - Click **Power button** (top left)
   - Status changes to "RAGS Running"

2. **Voice Interaction**
   - Click **Mic button**
   - Speak: "Hello RAGS"
   - Wait 1-2 seconds
   - RAGS responds in Hindi/English
   - Auto-resumes listening

3. **Camera**
   - Click **Camera button** (top right)
   - Grant permission
   - Click capture
   - Preview and use

4. **Other Features**
   - **Chat:** Click chat icon
   - **Automate:** Click automate icon
   - **Schedule:** Click schedule icon
   - **Settings:** Click settings icon
   - **Command Palette:** Press ⌘K

### Mobile App

1. **Voice Recording**
   - Tap **Mic button** (center)
   - Speak your message
   - Tap stop
   - See response

2. **Camera**
   - Tap **📷 Camera** in quick actions
   - Grant permission
   - Tap capture button
   - Retake or use photo

3. **Navigation**
   - **Home:** Main screen
   - **Chat:** Conversations
   - **Auto:** Automation
   - **Settings:** Configuration
